# ArmStudio · 通用机器人控制与轨迹设计软件

一个纯前端、开箱即用的机器人仿真、轨迹设计与 **CAD 建模** 工具。支持**任意结构的机器人**（机械臂、四足、SCARA…），可**导入自有机型**、在软件里**搭建设计机器人**、用**全身姿态关键帧**设计运动，并一键导出 **Arduino / ROS** 代码。

## 快速开始

用浏览器打开 `index.html` 即可（无需安装，推荐 Chrome / Edge，首次需联网加载 Three.js）。

## 核心功能

### 🎬 动作模式 — 机器人轨迹设计

**多机型 & 导入**
- 内置：6 轴机械臂、四足机器人、SCARA 平面臂。左上角下拉切换。
- 导入：支持自定义 JSON 机型，以及**简化 URDF**。
- 导出：把当前机型（含改动）保存为 JSON。

**主流工业轨迹设计（MoveJ / MoveL / MoveC）**
- 每段运动可选三种指令：`MoveJ`(PTP 关节插补)、`MoveL`(末端走直线)、`MoveC`(末端走圆弧)。
- 每段独立设定**速度**与**加速度**，运动按**梯形速度曲线**做加减速。
- 全局**速度倍率**滑块整体调速。
- **TCP 轨迹可视化**：3D / 2D 中按瞬时速度**色阶着色**（蓝慢→橙快）。
- 自动估算**循环节拍**。

**拖拽设计运动**
- 3D 视图：拖动末端执行器球体，自动逆解；多末端机型可在右侧选择当前末端。
- 2D 俯视图：点击 / 拖动设定当前末端的 X/Z，配合高度滑块设定 Y。
- **目标位置面板**：直接键入末端目标 X/Y/Z 坐标并移动到该点。
- **🖱 点击取点模式**：在 3D 场景点击地面即可连续落下运动点，快速画出整条路径。
- **关节点击拖拽**：点击 3D 场景中的关节球体可直接拖拽改变关节角度，滑块同步高亮。
- **路点坐标编辑**：选中路点后显示 X/Y/Z 坐标输入框，可直接编辑修改。
- **固定运动轴**：锁定 X/Y/Z，拖动时该轴不变。
- 关节滑块：手动精调每个关节。

### 🛠 CAD 建模 — 从零搭建任意机器人

**结构设计**
- 切到「🛠 CAD 建模」进入工作台，可「📄 新建空白」从底座开始搭建。
- **添加可动部件**：选中父连杆后，一键添加旋转/移动关节、固定骨骼、末端工具。
- **结构树**：层级树点选/查看每个连杆。
- **操作手柄 Gizmo**：三轴箭头（平移）+ 三环（旋转）手柄，支持网格吸附。
- **数值精确输入**：原点 X/Y/Z、关节限位可直接键入。
- **3D 自由放置** 🎯：在 3D 场景点击地面直接放置方块/圆柱/球体/圆锥等零件。

**外形编辑**
- 支持 **10 种基本几何体**：方块、圆柱、球体、圆锥、圆环、棱锥、楔形、管材、齿轮、拉伸体。
- **拉伸体**支持截面：矩形、圆形、三角形、任意多边形。
- **多边形顶点编辑器**：可自由创建不规则形状。
- **挖孔 (Cutouts)**：支持圆形、方形、三角形孔洞。
- **草图绘制 v2** ✏️：三步工作流——①选择草图平面(XY/YZ/XZ)→②选择形状并设参数→③设定拉伸厚度。3D 场景中实时显示草图平面网格。

**🎨 色彩与材质**
- **调色板**：10 种预设颜色一键切换。
- **材质类型**：标准、金属、磨砂、光泽、自发光。
- **透明度滑块**：5%~100% 可调。
- **随机颜色 🎲**：一键生成。

**变换工具**
- **镜像**：沿 XY/YZ/XZ 平面镜像复制。
- **阵列**：线性阵列（数量/间距/方向）和圆周阵列（数量/角度）。
- **尺寸标注 📐**：显示选中零件的宽/深/高和坐标。

**⚡ 布尔运算**：合并(∪)、减去(⊖)、相交(∩) 三个拉伸体零件。

### 代码自动生成

- `Arduino`：N 路伺服（含夹爪自动加一路），带插值平滑运动。
- `ROS (Python)`：发布 `trajectory_msgs/JointTrajectory` 的 rospy 节点。
- `轨迹 JSON`：通用导出，含每帧关节值与夹爪状态。

## 机器人定义格式（自定义 JSON）

机器人是一棵节点树，每个节点 = 连接到父节点的关节 + 连杆。角度单位**度**，长度 **mm**：

```json
{
  "name": "我的机器人",
  "type": "arm",
  "hasGripper": true,
  "home": [0, -20, 40],
  "links": [
    { "name": "base", "parent": -1,
      "joint": { "type": "revolute", "axis": [0,1,0], "origin": [0,0,0], "limit": [-180,180] },
      "geometry": { "type": "cylinder", "size": [50,30] } },
    { "name": "J2", "parent": 0,
      "joint": { "type": "revolute", "axis": [0,0,1], "origin": [0,70,0], "limit": [-110,110] } },
    { "name": "tip", "parent": 1,
      "joint": { "type": "fixed", "origin": [0,150,0] }, "endEffector": true }
  ]
}
```

## 项目结构

```
index.html           页面与布局
css/style.css         样式
js/
  kinematics.js       通用机器人树模型 + 运动学 + CCD 逆解
  shapes.js           3D 形状系统（10种几何体 + 草图轮廓 + 布尔运算）
  presets.js          内置机型
  urdf.js             简化 URDF 解析
  orbit.js            轨道相机控制
  gizmo.js            平移/旋转操作手柄
  robot3d.js          Three.js 渲染 + 多末端拖拽 + 材质系统 + 尺寸标注 + 草图平面
  view2d.js           2D 俯视图 + 草图模式
  sketch.js           草图管理器
  motion.js           运动规划（MoveJ/L/C、梯形速度曲线）
  trajectory.js       姿态关键帧与播放
  designer.js         结构编辑 + 镜像/阵列
  partslib.js         标准零件库（7大类30+种工业零件）
  codegen.js          代码生成
  examples.js         示例运动
  app.js              主控制器
```

## 技术说明

- 渲染：Three.js (r128, CDN)。逆运动学：CCD（循环坐标下降）。
- 坐标系 Y 轴向上；内部旋转用弧度，显示/导出为度。
- 外形几何使用 Three.js 原生 BoxGeometry/CylinderGeometry/ConeGeometry/TorusGeometry/SphereGeometry/ExtrudeGeometry。
- 标准零件使用 Group 组合几何体，支持完整参数化。
